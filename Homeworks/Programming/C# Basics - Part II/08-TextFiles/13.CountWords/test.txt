In this chapter we will review how to work with text files in C#.
We will explain what a stream is, what its purpose is, and how to use it.
We will explain what a text fileis and how can you read and write data to a text file
and how to deal with different character encodings.
We will demonstrate and explain the good practices for exceptionhandling when working with files.
All of this will be demonstrated with many examples in this chapter.
